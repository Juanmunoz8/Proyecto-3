# Proyecto 3

# Descripción
Este proyecto funciona como un simulador de Máquinas de Turing de una cinta usando el lenguaje Python.  
El simulador:
- Lee la definición de una MT desde un archivo YAML.
- Simula las cadenas de entrada configuradas.
- Muestra las descripciones instantáneas paso a paso.
- Indica si la cadena es (ACEPTADA) o (RECHAZADA).
